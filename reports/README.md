# Reports

Generated outputs from search and filtering runs.

## Typical Files

- `*_search_report.md` per university
- `*_filter_report.md` per university
- `final_combined_report.md` and `final_combined_report_latest.md`

## Regeneration

Use `python3 -m api_evaluation.run_all_universities ...` and/or individual filter scripts to refresh report content.
